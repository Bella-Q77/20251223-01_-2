# 幼儿园圣诞树生成系统

一个基于Flask的幼儿园圣诞树生成Web应用，支持管理员登录、小朋友姓名输入、圣诞树生成和数据保存功能。

## 功能特性

- **管理员登录**：通过指定账号和密码登录系统
- **姓名输入**：支持输入小朋友姓名生成专属圣诞树
- **圣诞树生成**：根据输入的姓名生成带有姓名的圣诞树
- **数据保存**：自动保存生成的圣诞树数据
- **数据调取**：支持再次输入相同姓名时回显已保存的圣诞树
- **童趣化设计**：符合幼儿审美的色彩明快、活泼可爱的页面风格

## 技术栈

- **后端**：Python Flask
- **前端**：HTML5, CSS3, JavaScript
- **数据存储**：JSON文件

## 安装和运行

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 运行应用

```bash
python app.py
```

### 3. 访问应用

打开浏览器访问：`http://localhost:5000`

## 使用说明

### 管理员登录

- 账号：`test`
- 密码：`admin`

### 生成圣诞树

1. 登录系统后，在"输入小朋友姓名"页面输入小朋友的名字
2. 点击"生成圣诞树"按钮，系统会生成带有该小朋友姓名的圣诞树
3. 系统会自动保存数据，再次输入相同姓名时会回显已保存的圣诞树

### 查看已保存的圣诞树

- 在页面下方的"已保存的圣诞树"区域可以查看所有已保存的小朋友姓名
- 点击姓名可以快速填入输入框

## 项目结构

```
.
├── app.py                  # 主应用文件
├── templates/              # HTML模板目录
│   ├── login.html         # 登录页面
│   └── christmas_tree.html # 圣诞树生成页面
├── requirements.txt        # 依赖文件
├── README.md              # 项目说明文档
└── christmas_trees.json    # 数据存储文件（自动生成）
```

## 页面设计

- **登录页面**：粉色渐变背景，雪花动画，圣诞树装饰
- **圣诞树生成页面**：紫色渐变背景，雪花动画，左侧输入区，右侧圣诞树展示区
- **已保存姓名区**：底部显示所有已保存的小朋友姓名，可点击选择

## 数据存储

生成的圣诞树数据保存在`christmas_trees.json`文件中，格式如下：

```json
{
    "张三": {
        "name": "张三"
    },
    "李四": {
        "name": "李四"
    }
}
```

## 开发说明

- 应用使用Flask框架开发，支持热重载（debug模式）
- 前端使用纯HTML/CSS/JavaScript实现，不依赖任何前端框架
- 数据存储使用JSON文件，简单易用
